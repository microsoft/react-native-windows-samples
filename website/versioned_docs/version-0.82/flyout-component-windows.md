> **Removed:** The Flyout component was part of the Old Architecture (Paper) and has been removed in React Native Windows 0.82. All apps now run exclusively on the [New Architecture (Fabric)](https://reactnative.dev/architecture/landing-page). For previous documentation, see the [0.81 version](https://microsoft.github.io/react-native-windows/docs/0.81/flyout-component).
